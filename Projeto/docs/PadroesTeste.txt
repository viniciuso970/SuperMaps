Arquivos de classes:
- Vao ficar na pasta de classes
- Terão formato *.class.php

Arquivos de template:
- Ficam na pasta view
- Extensão *.tpl

Como utilizar um template:
Se tivermos uma tabela, o número de linhas varia com o BD;
Vamos pegar criar a estrutura da tabela e colocar uma "variável" (será uma tag do tipo: [@tag]);
nessa variável iremos substituir todos os dados do BD, como cada dado vai ser uma linha, vamos criar um outro template;
nesse template irá ter o formato da linha, com outras tags, substituindo cada uma com os valores do BD;
